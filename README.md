# Inventory-Dashboard
Project Overview : This project is an interactive Power BI dashboard designed to analyze retail inventory performance and sales trends across time, product categories, and geographic regions. It enables better visibility into inventory efficiency and supports data-driven decision-making.

# Features:
Utilized Power Query to perform comprehensive data cleaning and transformations, including validating data types and removing inconsistencies from raw data files to ensure high-accuracy reporting



Built an interactive dashboard with an advanced user interface by developing a slide-out filter pane using bookmarks and transparent overlays to let users monitor retail inventory performance across time, product categories, and geographical regions





Performed advanced DAX calculations to dynamically track key sales and supply chain KPIs, including Sell-Through Rates, Inventory Turnover, and conditional stock-status alerts
